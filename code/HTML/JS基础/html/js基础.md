二

变量：
定义没有赋值初值，这个默认值为undefined;一个变量只有被var定义，并赋值初值之后，才算正式初始化成功；
var a;
console.log(a);
打印undefined;
定义变量不加var，就会产生作用域的问题；
同时声明多个变量：
    var a=0,b=0;
变量声明提升：
    提前使用一个稍后才声明的变量，不会引发异常；因为预解析阶段的原因，可以先使用再定义；
