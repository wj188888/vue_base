盒模型的基本概念：
    所有的HTML标签都可以看成举行盒子：由width,height,padding,border,称为盒模型;

    width是指内容的宽度
    height是指内容的高度

    文字到边框的距离是内边距
盒子总宽度=width+左右padding+左右border
border: 10px solid #000;/* 宽度，实线，颜色 */

二

width属性px，移动段使用百分数，rem等
当块级元素（div,h,li等）没有设置width属性时，它将自动撑满
height属性px，不设置默认为0；

三

padding是盒子的内边距
四个方向的padding，可以分别使用小属性进行设置
padding-top
padding-right
padding-bottom
padding-left
连写：
    上右下左
三连写：
    上（右左） 下
两连写：
    （上下）（左右）
    
不写的一条边0px,可以写出0；
padding: 40px;
padding-bottom: 0;
用小属性层叠大属性；
